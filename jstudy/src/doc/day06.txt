제어명령
==> 컴퓨터는 원칙적으로 코딩한 순서대로 실행함
	하지만 프로그램을 만들다보면 이 순서를 바꿔야하는 경우가 발생할 때 사용하는 명령
	
	1. 조건 제어 명령
		==> 특정 조건에 따라 실행순서를 변경하는 명령
		
		형식 ] 
			if(조건식){
			
			}[[else if(조건식){
			
			}else if(조건식){
			
			}
			...]
			else{
			
			}]
		
		형식 1 ]
			if(조건식){
				실행내용
			}
			
			해석 ] 조건식이 참이면 실행내용을 실행, 거짓이면 스킵.
			
	참고 ] 
		자바의 명령을 보면 {} 이 존재하는 경우를 "블럭마크"라고 표현한다.
		실행범위를 지정하는 것을 의미한다.
		
		}는 ;을 대신할 수 있다.
		
	참고 ] 
		{}안에 실행내용이 오직 한 문장이면 {}를 생략가능 BUT 되도록이면 쓰도록 한다.
		
		형식 2 ]
			if(조건식){
				실행내용1
			}else{
				실행내용2
			}	
			
			의미 ] 
				조건식이 참이면 실행내용1을 실행, 거짓이면 실행내용2를 실행.
		
		형식 3 ] 
			if(조건식1){
				실행내용1
			}else if(조건식2){
				실행내용2
			}else if(조건식3){
				실행내용3
			}
			...
			else{
				실행내용n
			}
			
			의미 ] 조건식1이 참이면 실행내용1을 실행, 거짓일경우 조건식2를 검색하고 참일경우 실행내용2, 거짓일경우 조건식3을 검색...모두거짓이면 실행내용n
			
			참고 ]
				조건식 2는 조건식1이 거짓인 경우에만 읽게된다.
				
			참고 ]
				if 명령도 중첩이 가능합니다. 
				
				if(조건식){
					...
					if(조건식){
						...
					}
				}else {
					if(조건식){
						...
					}else if(조건식){
						...
					}else{
						...
					}
				}
